# molssi-workshop
This covers the MolSSI Workshop materials and notes for python programming for bioinformatics and chemistry students.
MolSSI Workshop

This repository contains materials, exercises, and code developed during the MolSSI (Molecular Sciences Software Institute) Workshop.

Overview:

The purpose of this workshop is to introduce best practices in:

Version control with Git

Using GitHub for collaboration

Writing clean and maintainable code

Reproducible research workflows

Basic scripting and computational tools

This repository serves as a record of workshop activities and completed exercises.

📂 Contents

MolSSI Workshop.ipynb – Jupyter Notebook with workshop exercises and notes

Additional scripts and practice files (if applicable)
